# java-web-maven-quarkus-qute-hello-world

## Description
A POC for quarkus framework
using jsp-like template engine,
qute.

## Tech stack
- java
- maven
  - quarkus
    - jsp

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
